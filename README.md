## Overview

Personnel Verification – Staff Bio-Data App is a fully portable PHP-based application designed to manage staff bio-data records including passport photographs and fingerprint uploads.

The system uses **SQLite database (local file-based storage)** and requires no MySQL, XAMPP, or WAMP installation.

---

## System Requirements

- Windows PC
- PHP 7+ (with pdo_sqlite and sqlite3 enabled)
- Modern web browser (Chrome, Edge, Firefox, Opera)

> No MySQL required  
> No XAMPP required  
> Fully self-contained system

---

## Database Setup (Automatic)

On first run, the system automatically:

- Creates `data/` folder
- Creates `data/staff_biodata.sqlite`
- Creates `staff` table with required fields
